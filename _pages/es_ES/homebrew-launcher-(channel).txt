---
title: "Homebrew Launcher (Canal)"
---

Esto instalará el Homebrew Channel para que, una vez que tengas una SysNAND modificada funcionando, el Homebrew Launcher aparezca como un icono en el menú del sistema.

#### Instrucciones

1. Enter the Homebrew Launcher
  + CBHC users must press (Home) while CBHC is booting to access the CBHC boot options menu, then select the Homebrew Launcher
  + Haxchi users must launch Haxchi once to patch SysNAND, then run it a second time while holding (A) to launch the Homebrew Launcher
  + Mocha CFW users must launch Mocha CFW, then relaunch the Homebrew Launcher
1. Inicia WUP Installer GX2
  + Note that if WUP Installer GX2 crashes with a "DSi Exception Has Occurred" error, you should just shut down the console and try again.
1. Selecciona el Homebrew Channel
1. Presiona "Install", luego presiona "Yes" para confirmar
1. Selecciona "NAND" como destino
1. Selecciona WUP Installer GX2
1. Presiona "Install", luego presiona "Yes" para confirmar
1. Si planeas instalar juegos en un disco duro USB, selecciona "USB" como destino; en caso contrario, selecciona "NAND" como destino
1. Presiona (Home), luego cierra el programa para salir del instalador una vez que haya terminado

___

Ahora puedes iniciar el Homebrew Launcher en la SysNAND modificada simplemente iniciando el Homebrew Channel.
{: .notice}

Ahora puedes lanzar WUP Installer GX2 en la SysNAND modificada solamente iniciándolo del menú del sistema.
{: .notice}

Para mayor seguridad, *debes* crear un respaldo de tu consola siguiendo [Respaldo de la NAND](nand-backup).
{: .notice--primary}

Ten en cuenta que para acceder a la eShop con los las direcciones DNS que bloquean las actualizaciones, antes debes iniciar NNU-Patcher desde el Homebrew Launcher (NNU-Patcher es un parche temporal y tendrás que iniciarlo después de cada reinicio para entrar a la eShop).
{: .notice--info}

Ten en cuenta que, debido a un error, WUP Installer GX2 sólo puede realizar instalaciones en memorias USB (como un USB HDD) si mueves WUP Installer GX2 al almacenamiento USB usando el menú Gestión de datos de las Configuraciones de la consola.
{: .notice--warning}

Para informarte acerca de cómo volcar los datos de tus discos de juego a un formato instalable para jugar sin necesidad del disco, echa un vistazo al apartado de [disc2app](disc2app).
{: .notice--success}

Para informarte acerca de cómo modificar la vWii en SysNAND, echa un vistazo al apartado de [Modear la vWii](vwii-modding).
{: .notice--success}

Para informarte acerca de cómo desinstalar el CFW de forma segura y regresar la consola a su estado original, echa un vistazo al apartado de [Desinstalar CFW](uninstall-cfw).
{: .notice--warning}
